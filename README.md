# Green AI Efficiency Lab 🌿💻

## English Version
### Overview
This project is an experimental laboratory focused on Green IT and Sustainable Computing. It demonstrates the direct correlation between algorithmic complexity and energy consumption. By comparing Linear Search with Binary Search, we measure how optimized logic can drastically reduce CPU execution time and memory peak, contributing to a lower carbon footprint in large-scale data processing.

### The "Green" Difference
- High Energy Consumption: Sub-optimal algorithms keep processors at high load for longer.
- Sustainable Approach: Optimized algorithms save hardware life and energy.

## Versão em Português
### Visão Geral
Este projeto é um laboratório experimental focado em TI Verde e Computação Sustentável. Ele demonstra a correlação direta entre a complexidade algorítmica e o consumo de energia. Ao comparar a Busca Linear com a Busca Binária, medimos como a lógica otimizada reduz o tempo de execução da CPU e o pico de memória, diminuindo a pegada de carbono.

### O Diferencial Verde
- Alto Consumo: Algoritmos ineficientes mantêm o processador em carga alta por mais tempo.
- Abordagem Sustentável: Algoritmos otimizados preservam a vida útil do hardware e economizam energia.
